# AI Learning Lab

> A long-term journey to understand Artificial Intelligence, Machine Learning, Neural Networks, Transformers, Large Language Models, and AI Systems from first principles.

---

## Mission

This repository is not about completing tutorials.

It is about understanding how modern AI systems work by building them from scratch, one concept at a time.

Every project is accompanied by:

- Theory
- Mathematics
- Implementation
- Reflection
- Experiments
- Interview preparation

The goal is not to memorize APIs.

The goal is to understand why the technology exists, how it works internally, and when to use it.

---

## Learning Philosophy

We follow a bottom-up approach.

Understand the machine before using the machine.

Every topic answers:

- What is it?
- Why was it introduced?
- What problem does it solve?
- How does it work?
- How is it implemented?
- What are its limitations?
- Where is it used?
